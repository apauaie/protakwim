# Protakwim Raspberry Pi Prayer Time
Raspberry pi with Malaysia prayer time

This project is an extension of Raspberry Pi Prayer time from https://github.com/shafiqsaaidin/pi-prayertime


Demo Site can be accessed from here https://masjidpulaiindah.com/protakwim/

Installation

1) Download and Copy to server location ( for example : /var/www/html for Raspberry Pi Apache (Please install apache first)
2) Setup MySQL /PHPMyadmin new database and user
3) Navigate to the administrator folder from your Raspberry Pi/Server, for example http://localhost/administrator
4) Complete the installation procedure

5) Once installation is complete, you will see the initial page at http://localhost . You can edit other settings at http://localhost/administrator
Please ensure you select the correct location and update(kemaskini) the solat time with the latest time by cliking kemaskini.Make sure internet is available during kemaskini. 
You can also select the theme, background image, countdown important dates and others settings in the administrator page.
